## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rynkovski/mediporta-task.git
   ```
2. Install NPM packages
   ```sh
   npm ci
   ```
3. Start
   ```sh
   npm start
   ```
4. Open Storybook
   ```sh
   npm run storybook
   ```
